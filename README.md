# MusicPlayer

### Installation
```
npm install
```
### Running the server
```
nodemon server.js
```
 <p>In this project I used Node.js -  javaScript Html and CSS</p>
 <p>1. You need to register from the registration page</p>
 <p>2. Then you go login page</p>
 <p>3. Clik on search bar for all music</p>
